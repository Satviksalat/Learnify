# Machine Learning Quick Reference

## Types of Learning
1. **Supervised**: Input data is labeled (e.g., Regression, Classification).
2. **Unsupervised**: Input data is unlabeled (e.g., Clustering).
3. **Reinforcement**: Learning via rewards/punishment.

## Common Libraries
- **NumPy**: Math & Arrays
- **Pandas**: Data DataFrames
- **Matplotlib**: Plotting & Graphs
- **Scikit-learn**: ML Algorithms
- **TensorFlow**: Deep Learning
- **Keras**: Neural API

## ML Workflow
1. **Data Collection**: Gathering raw data.
2. **Preprocessing**: Cleaning, Scaling.
3. **Split**: Train/Test split.
4. **Training**: `model.fit(X_train, y_train)`
5. **Evaluation**: `model.predict(X_test)`
