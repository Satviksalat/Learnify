# Python Cheat Sheet

## Key Data Types
- **int**: Integer (1, -5)
- **float**: Decimal (1.5, 3.14)
- **str**: String ("Hello")
- **bool**: Boolean (True, False)
- **list**: [1, 2, 'a'] (Mutable)
- **tuple**: (1, 2) (Immutable)
- **dict**: {'a': 1} (Key-Value)

## String Methods
- `s.lower()`: Lowercase
- `s.upper()`: Uppercase
- `s.strip()`: Remove whitespace
- `s.replace(a, b)`: Replace a with b
- `s.split(sep)`: Split string into list
- `len(s)`: Length of string

## List Operations
- `lst.append(x)`: Add x to end
- `lst.pop()`: Remove last item
- `lst.sort()`: Sort list
- `lst[i]`: Access element at i
- `x in lst`: Check membership
- `len(lst)`: Count items

## Control Flow
```python
if condition:
    # code
elif other:
    # code
else:
    # code

for i in range(5):
    print(i)

while x < 10:
    x += 1
```
